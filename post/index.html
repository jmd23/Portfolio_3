January 11, 2022

Netflix-Lite

Built an app using ReactJS, CSS, and HTML. Modeled the frontend after Netflix, including rows of movie posters (first image below), a large picture at the top of the page, and the ability to scroll. Uses the TMDB movies API to get data for the app, which is hosted by Firebase. This project also included a data science component, as I built a movie recommendation system Jupyter Notebook using Python and Scikit-Learn, which can be navigated to from the main page.
December 23, 2021

Sports Analytics: Baseball

This was a semester-long project for the Cornell Sports Analytics club. We looked at how external factors (crowd speed, temperature, attendance), affect intrinsic components of a baseball game (Runs Scored, Duration, etc…), using data science techniques and Python libraries, such as Pandas and MatPlotLib, to produce visualizations and find correlations. I also implemented machine learning models to predict the winner of a game, with some reaching 80% accuracy (first image). The work was presented at the club’s final symposium.
May 10, 2022

BearMap

This was a project I completed during my research internship at Brookhaven National Lab, for the Regeneron STS competition. It was done in collaboration with Dr. Laura Fierce, and involved a mix of environmental science and data analysis aspects. In short, I wrote python scripts to determine what effect surfactants have on the formation of cloud nuclei, as aerosol-cloud interactions are currently one of the least understood aspects of climate modeling.
