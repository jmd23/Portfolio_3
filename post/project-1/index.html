Sports Analytics: Baseball

December 23, 2021

This was a semester-long project for the Cornell Sports Analytics club. We looked at how external factors (crowd speed, temperature, attendance), affect intrinsic components of a baseball game (Runs Scored, Duration, etc…), using data science techniques and Python libraries, such as Pandas and MatPlotLib, to produce visualizations and find correlations. I also implemented machine learning models to predict the winner of a game, with some reaching 80% accuracy (first image). The work was presented at the club’s final symposium. Some components include:

  • Worked with Kaggle datset; cleaned Na values and created new dataframe columns to better categorize data
  • Produced correlation matrix with pandas, detailing all variables (second image)
  • Made individual scatterplots with MatPlotLib to observe certain correlations, for example runs scored vs. temperature
  • Implemented machine learning models from scikit-learn (Linear, Naive Bayes, SVM, etc..) and measured accuracy after splitting train/test at 80:20 ratio
